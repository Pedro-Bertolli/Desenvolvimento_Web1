Elementos Semânticos

<article>
Representa um conteúdo independente, como uma postagem de blog ou artigo, que pode ser entendido por si só
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_article
<header>
Define a área de cabeçalho de uma página ou seção, geralmente contendo títulos, logotipos e menus de navegação
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_header
<footer>
Define um rodapé para um documento ou seção.
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_footer
<nav>
Define um conjunto de links de navegação.
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_nav
<aside>
Define algum conteúdo além daquele em que é colocada.
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_aside
<section>
Define uma seção em um documento.
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_section
<details>
Especifica detalhes adicionais que o usuário pode abrir e fechar sob demanda.
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_details
<main>
Especifica o conteúdo principal de um documento.
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_main
<summary>
Define um título visível para o elemento <details> . O título pode ser clicado para visualizar/ocultar os detalhes.
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_summary
<figcaption>
Define uma legenda para um elemento <figure>.
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_ficaption
<mark>
Define o texto que deve ser marcado ou destacado.
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_mark
<time>
Define um horário específico (ou data/hora).
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_time
<figure>
Especifica conteúdo independente, como ilustrações, diagramas, fotos, listagens de códigos, etc.
Exemplo: https://www.w3schools.com/html/tryit.asp?filename=tryhtml5_figure
